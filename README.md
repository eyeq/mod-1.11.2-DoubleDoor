# [両開きドア]DoubleDoor
## 本家：kirsybuu 様
[[1.5.1] Double Door Mod v4.6 [Forge]](http://www.minecraftforum.net/forums/mapping-and-modding/minecraft-mods/1274626-1-5-1-double-door-mod-v4-6-forge)

[**ダウンロード**](https://github.com/eyeq/mod-1.11.2-DoubleDoor/releases/download/1.0/1.11.2-DoubleDoor-1.0.jar)

扉を開けたときに、接している扉を連動させます。  
連動する方向は、以下の通りです。  
ドアは取っ手の方向、合計1方向  
トラップドアはxz平面で自身を中心に3×3、合計8方向  
フェンスゲートは上下左右の十字形、合計4方向
